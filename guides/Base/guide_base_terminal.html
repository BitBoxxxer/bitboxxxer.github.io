Введение

Этот гайд нужен для базового ознакомления с терминалом и его командами ОС Windows

Что тут забыл Git ?

Ранее, в Base_#1 уроке по основам Git - мы разобрали что исполняемые команды можно выполнять в git Bash терминале.
Но так же все команды git можно использовать и в обычном терминале вашей ОС или терминале в вашей IDE (Например Rider или VSC).

Все терминалы - одинаковы ?

Можно скаать и так. Но в любом случае есть разница - ограничениях прав пользователя к тем или иным действиям.
Допустим для исполнения некоторых команд - вам потребуется запуск терминала с правами администратора. (Что можно сделать через ПКМ - открыть от администратора).
