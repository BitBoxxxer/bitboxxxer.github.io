Введение

Краткое введение в тему гайда. Что будет рассмотрено и для кого этот гайд.

Раздел 1: Название раздела

Основной контент гайда. Разбивай на логические разделы для удобства чтения.

Пример кода

Описание примера:

язык 
// Твой код здесь
// Пример:
function example() {
    return "Hello, World!";
}

Раздел 2: Пошаговая инструкция

Разбивай сложные процессы на шаги:

  1. Первый шаг - описание первого шага. Можно использовать inline код для выделения.
  2. Второй шаг - описание второго шага.
  3. Третий шаг - описание третьего шага.
